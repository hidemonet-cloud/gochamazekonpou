# ごちゃまぜ梱包シミュレーター　iPad設定手順

## 必要なファイル
- index.html
- manifest.json
- sw.js
- icon-192.png（任意）
- icon-512.png（任意）

---

## ステップ1: GitHub Pages に公開する（無料・5分）

### 1-1. GitHubアカウント作成
https://github.com にアクセスして無料アカウントを作成

### 1-2. 新しいリポジトリを作成
- 右上の「+」→「New repository」
- Repository name: `konpou-sim`（任意）
- Public を選択
- 「Create repository」をクリック

### 1-3. ファイルをアップロード
- 「uploading an existing file」をクリック
- index.html / manifest.json / sw.js をドラッグ＆ドロップ
- 「Commit changes」をクリック

### 1-4. GitHub Pages を有効化
- リポジトリの「Settings」タブ
- 左メニュー「Pages」
- Source: 「Deploy from a branch」
- Branch: 「main」→「/(root)」
- 「Save」をクリック

### 1-5. URLを確認
数分後に以下のURLでアクセス可能になります：
```
https://[あなたのユーザー名].github.io/konpou-sim/
```

---

## ステップ2: iPadのホーム画面に追加する

1. iPadのSafariで上記URLを開く
2. 画面下部の「共有ボタン」（□↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 名前を確認して「追加」をタップ

→ ホーム画面に「梱包SIM」アイコンが追加されます！

---

## 使い方
- アイコンをタップして起動（フルスクリーンで表示）
- 商品の＋ボタンで数量を入力
- リアルタイムで3D確認
- Wi-Fi環境で一度開けば、次回からオフラインでも使用可能
